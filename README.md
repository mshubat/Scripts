## Scripts
* **enthusiast_achievement** - helps the stack overflow user to get the "enthusiast achievement": given for visiting the site 30 consecutive days
* **Hourglass** - (HackerRank Challenge) finds the largest hour glass present in a 6x6 Matrix (whose values fall between -9 to 9 inclusive) and prints out its sum.
* **pingHosts** - Pings a series of addresses on a network. Based on the number of replies received, it can be used to estimate the number of hosts on the network (Assumes Classfull Addressing).
